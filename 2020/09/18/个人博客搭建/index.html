个人博客搭建

​ 秋招马上结束，想着自己什么都还不会，于是起了赶几个项目的念头，希望通过实战项目，让我的简历变得丰富起来，同时在此过程中，整合这几年所学的零零碎碎的知识。和同学商量着做个聊天室的项目，但这几天他有事，时间空余，正好手里头有服务器和域名，于是利用这段时间搭个个人博客练练手。

​ 在B站关注了个UP：CodeSheep，本文章参照他的视频教程攥写，记录本人的实际搭建过程。视频链接：https://www.bilibili.com/video/BV1Yb411a7ty/

安装配置Hexo

​ Hexo是一个基于nodejs的静态博客网站生成器。

特点：

  • 不可思议的快速 ─ 只要一眨眼静态文件即生成完成
  • 支持 Markdown
  • 仅需一道指令即可部署到 GitHub Pages 和 Heroku
  • 已移植 Octopress 插件
  • 高扩展性、自订性
  • 兼容于 Windows, Mac & Linu
